# Lernperiode-8 Three.js

## 9.1.26

Von den Vorschlägen sprachen mich Three.js, Rust und TypeScript an. Rust und TypeScript, da ich schon etwas über sie gehört habe, und Three.js weil es mich interessiert, wie man 3D‑Objekte mit Code erstellen kann. Rust fiel schnell aus der Auswahl heraus, da es mir zu schwer erschien und überall stand, dass es schwieriger ist als viele andere Sprachen. Ich ging also auf die Suche nach Three.js und fand diese Website: https://threejs.org/ . Auf der Website fand ich viele coole Projekte, die mit Three.js gemacht wurden, wie dieses hier: https://exp-gemini.lusion.co/motion oder das hier: https://www.anumberfromtheghost.com/ . Ich hatte mich also schon ziemlich entschieden. Nur um sicherzugehen, suchte ich trotzdem nochmal nach TypeScript, aber fand nichts, was mich überzeugte.

Als Erstes suchte ich mir ein Tutorial heraus für den Einstieg in die Arbeit mit der JS-Bibliothek: https://www.youtube.com/watch?v=XPhAR1YdD6o 
Im Tutorial wird gut erklärt, was man macht, und das habe ich einigermassen verstanden. Ich habe das Projekt also nachprogrammiert und noch selber ein paar kleine Sachen verändert zum Herumprobieren. Es ist also nicht mehr 1:1 wie im Tutorial. Ich musste mir dafür noch den Code von diesem Github herunterladen: https://github.com/bobbyroe/getting-started-with-threejs .
